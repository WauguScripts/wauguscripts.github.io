WinFix Versie 1.1 (18-11-2025)

Wat het is:
Een Batch script, gecompileerd naar een .exe bestand.

Wat het doet:
WinFix kan helpen om fouten te herstellen met DISM (Deployment Imaging Service and Management tool) en SFC (System File Checker).

DISM  is een hulpprogramma dat standaard aanwezig is in Windows. Het controleert het Windows image-bestand op fouten en repareert deze indien nodig.
Als bepaalde systeembestanden ontbreken, is een internetverbinding vereist om ze opnieuw te downloaden.

SFC  is eveneens standaard beschikbaar in Windows. Deze tool controleert de integriteit van systeembestanden en herstelt eventuele beschadigingen met behulp van gegevens uit het Windows image.

Beide tools zullen achter elkaar worden uitgevoerd. Dit kan helpen bij het opsporen en herstellen van fouten in het Windows-systeem.
Gebruik dit script als Windows onverklaarbare fouten maakt.

Laatste wijziging: Veiligheid toegevoegd en uitvoer in Nederlands.
Download      Portable    

ScreenSwitch 1.1 (22-12-2025)

Wat het is:
Een simpele app, geschreven in AHK 1.2.

Wat het doet:
Het zet 2 icons op je bureaublad . Deze kan je aan de taakbalk pinnen, zodat je snel van bureaublad 1 naar bureaublad 2 en terug kan switchen.

Over de portable versie:
Plaats je de bestanden op het bureaublad dan zullen de snelkoppelingen niet meer werken als deze bestanden worden gewist. Bij voorkeur moeten de bestanden worden uitgepakt naar een andere map (bijvoorbeeld C:\Script\ScreenSwitch). vanuit deze map kan de app worden gestart waarna de iconen op het bureaublad worden geplaatst. Deze kan je aan de taakbalk of startmenu pinnen. Er zijn dus meerdere handelinhen nodig.
De Full Install versie heeft het voordeel dat alles in 1x goed wordt geinstalleerd. Mocht je de iconen hebben verwijderd dan zijn ze direct terug te plaatsen door in het startmenu te klikken op "Set_ScreenSwitch".

Download        Portable

FolderMirror 1.0 (13-12-2025)

Wat het is:
Een batch script, gecompileerd naar een .exe bestand

Wat het doet:
Maakt een backup van een gekozen map naar een gekozen doelmap, inclusief submappen.
Tijdens de eerste synchronisatie worden alle bestanden volledig gekopieerd. Bij daaropvolgende synchronisaties worden uitsluitend nieuwe of gewijzigde bestanden overgezet. (Tijdwinst)
Bestanden die in de bronmap zijn verwijderd, worden eveneens uit de doelmap verwijderd. Hierdoor blijft de doelmap een exacte weergave van de bronmap. Dit proces wordt aangeduid als spiegelen.

Voor een uitgebreidere grafische versie (windows venster, muisbesturing) download DocuSafe.
Download      Portable    

DiskInit 1.0 (12-12-2025)

Wat het is:
een app, geschreven in AHK 1.1
De opvolger van PartitionSwitch.

Wat het doet:
Dit script zet de partitie indeling van een HDD, SSD of USB-stick om naar een andere indeling.
Je hebt de keuze tussen GPT (Guid Partition Table) en MBR (Master Boot Record).

Dit script werkt alleen bij geformatteerde schijven met een drive-letter.
Een ongeformatteerde schijf of een schijf zonder letter toewijzing kunnen worden aangepast via Windows Schijfbeheer.

Laatste wijziging: App opnieuw geschreven: Nu met grafische schermen en muisbesturing.
